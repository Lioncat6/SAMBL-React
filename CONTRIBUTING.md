# Contribution to SAMBL
All contributions are welcome, feel free to contribute to code or documentation!

## Code submissions
Please submit PRs to the `dev` branch, including documentation

## Contributing Guidelines
- Please check the current issues and pull requests before submitting any changes
- **Fully AI/LLM generated code is completely dissalowed, and any applicable Pull Requests will be immediately closed**
- AI assisted code usage should be declared in pull requests and may still be denied
- Any applicable code licenses should be mentioned in pull requests
